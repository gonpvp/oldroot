## Installing

1. Installing Python (at least version 3.7.8)
2. Clone the repo
   ```sh
   git clone https://github.com/gonpvp/oldroot
   ```

### Use Script Grabber

1. Start file run.bat
2. Type the command
   ```
   main.py
   ```
3. Type the command
   ```
   help
   ```
4. And you will have the whole list of possible orders!
